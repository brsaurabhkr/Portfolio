# Portfolio

A simple personal portfolio website showcasing projects, skills, and contact info.

## About
This repository contains the source for a personal portfolio site to present projects and professional details.

## Usage
- Clone the repo: `git clone https://github.com/brsaurabhkr/Portfolio.git`
- Open `index.html` in your browser or serve the folder with a static server (e.g., `npx serve`).
